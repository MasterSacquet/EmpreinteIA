# Calcul

Le modèle combine :
- Énergie terminal
- Énergie réseau
- Énergie data center
- Type de modèle IA
- Mix électrique du pays

## Concepts clés
- Analyse du cycle de vie (ACV)
- Empreinte carbone

---

## Empreinte carbone

CO2e = (Eterminal + Ereseau + Edata) × CI

### Variables :
- Eterminal : énergie du device utilisateur (partie 3)
- Ereseau : énergie du transport de données (partie 6)
- Edata : énergie des serveurs IA (partie 4)
- CI : intensité carbone (kgCO₂e/kWh, dépend du pays)

### Détail des calculs

#### Calcul par jour :

1) Eterminal_jour :

Eterminal_jour = P_device × t_usage_total

t_usage_total = t_usage_texte + t_usage_image + t_usage_video + t_usage_audio + t_usage_code

Pour texte, image, vidéo, audio : t_usage_X = N_requete x t_requete 

Pour code : t_usage_code = "réponse à la question "temps d'utilisation"

2) Ereseau_jour :

Ereseau_jour = Data_total × EI_reseau 

Data_total = Data_texte + Data_image + Data_video + Data_audio

Pour texte, image, video et audio : Data_X = GB_par_requete x N_requete

3) Edata_jour :

Edata_jour = Edata_texte + Edata_image + Edata_video + Edata_audio + Edata_code

Pour texte, image, video, audio et code : Edata_X = N_requete × E_requete x type_plan

En fonction de la réponse à "Version utilisée", type_plan = :
- Gratuit : 1
- Payant : 3
- Entreprise : 6

#### Calcul par mois :

En fonction de la réponse à "Fréquence d'utilisation", Frequence_utilisation = :

- Occasionnelle : 7
- Régulière : 15
- Intensive : 30

Eterminal_mois = Eterminal_jour x Frequence_utilisation

Ereseau_mois = Ereseau_jour x Frequence_utilisation

Edata_mois = Edata_jour x Frequence_utilisation

---

## Empreinte eau

Eau = (Eterminal + Ereseau + Edata) × WI  

- WI = intensité hydrique (L/kWh)
